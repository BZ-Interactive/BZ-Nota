BZ-Nota

BZ-Nota is a high-performance, feature-rich, terminal-based text editor built with C++ and FTXUI. Combining a modern UI with a tiny footprint—using less than 6MB of RAM—it offers a seamless experience for both raw text and Markdown editing.

🧠 <6MB RAM 📦 1.6MB binary ⚡ ~1% CPU
